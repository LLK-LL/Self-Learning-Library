# DOCX edit scope must be verified before manuscript changes

## Rule Type

- `process_governance`

## Governance Rule

Before editing a DOCX manuscript, verify whether the file is the full manuscript or a partial draft. Scope verification is a workflow guard and must not be treated as a manuscript-writing requirement.

## Boundary

- This rule governs the knowledge-base workflow only.
- It must not be treated as a manuscript claim, section-structure requirement, wording rule, or paper evaluation criterion.

## Evidence Memories

- [[DOCX manuscript edits must distinguish partial drafts from full manuscript]]

## Governance Test

Before a DOCX edit, record the target file scope as full manuscript or partial draft, then verify the output file still contains the expected sections.
