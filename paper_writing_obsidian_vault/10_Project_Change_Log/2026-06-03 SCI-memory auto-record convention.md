# 2026-06-03 SCI-memory auto-record convention

## Change Event

用户明确要求：当启用 `SCI-memory` 后，后续应自动记录每次对话中的论文修改过程和用户提出的建议。

## Implementation Meaning

该约定改变后续论文写作任务的记录方式。以后凡是用户启用 `SCI-memory` 并进行论文相关修改、讨论导师意见或提出写作建议，应在任务完成后主动将关键过程写入论文知识库，优先记录到 `40_Paper_Memories`；若涉及具体稿件修改过程，则同步记录到 `10_Project_Change_Log`。

## Boundary

该约定不等于自动运行知识库迭代脚本，也不适用于非论文写作类任务。
