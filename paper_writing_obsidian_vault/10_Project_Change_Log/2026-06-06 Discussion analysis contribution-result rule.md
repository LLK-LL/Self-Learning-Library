# 2026-06-06 Discussion analysis contribution-result rule

## Change

The user corrected the draft discussion-analysis wording for the intermetallic-compound thickness section. The selected sentence framed the paragraph around prediction limitations, reaction time, diffusion coefficients, and phase-transformation kinetics. The user stated that this kind of content should not appear in Discussion/Analysis and that the section should mainly write this paper's contribution and results.

## Applied Rule

Future Discussion/Analysis writing in this manuscript should emphasize:

- results obtained in this paper;
- figure-supported evidence;
- the closed chain between temperature field, EDS characterization, and intermetallic-compound thickness prediction;
- the contribution of the SPH-FEM thermoplastic coupling model.

It should avoid ending ordinary discussion-analysis paragraphs with limitation-style caveats unless the user explicitly asks for a limitations paragraph.
