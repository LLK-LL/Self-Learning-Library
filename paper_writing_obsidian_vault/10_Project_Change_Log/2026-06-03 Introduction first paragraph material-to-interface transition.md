# 2026-06-03 Introduction first paragraph material-to-interface transition

## Change Event

用户指出，引言第一段中“TC4/Al6061 爆炸焊接复合板兼具钛合金的高强度和铝合金的低密度、吸能特性，在轻型装甲结构中具有应用潜力”与下一句“在爆炸焊接过程中，飞板与基板在高速碰撞作用下发生剧烈塑性变形……”之间存在跳跃，句子之间缺少连贯性。

## User Suggestion

两句话不能从材料性能优势直接跳到爆炸焊接过程现象，中间需要说明复合板性能发挥与界面可靠连接之间的关系，使材料体系、连接工艺和界面波问题形成连续推进。

## Revision Direction

在材料优势句之后增加或改写承接关系：TC4/Al6061 复合板的承载与吸能能力能否发挥，取决于钛/铝界面的可靠连接；爆炸焊接通过高速碰撞形成冶金结合，因此界面形成过程成为需要关注的核心环节。随后再引出剧烈塑性变形、材料流动和界面波形成。

## Rationale

该修改将段落推进关系由“材料优势 -> 工艺现象”调整为“材料优势 -> 界面连接要求 -> 爆炸焊接界面形成 -> 界面波问题”，避免句间断裂，同时扣合导师提出的每一句内容都应服务于论文主题的要求。

## User Correction And Priority Update

用户随后提供了更符合预期的最小改动版本：保留原有“TC4/Al6061 爆炸焊接复合板兼具钛合金的高强度和铝合金的低密度、吸能特性，在轻型装甲结构中具有应用潜力。”以及后续爆炸焊接过程句，只在二者之间加入“然而，高质量 TC4/Al6061 复合板的制造对爆炸焊接而言是一项挑战。”用户明确指出，之前的改写幅度过大且仍显口语化。因此，后续同类修改应提高“尽可能不动原文、只补必要过渡句”的权重。
