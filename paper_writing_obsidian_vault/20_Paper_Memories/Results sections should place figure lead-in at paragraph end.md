# Results sections should place figure lead-in at paragraph end

## Source Metadata

- Memory ID: `results-figure-leadin-2026-06-04`
- Type: `user-preference`

## Memory Content

For manuscript result sections in this project, when the user specifies that a chapter should only display results, avoid discussion-style analysis and mechanism explanation. Figure-introduction sentences such as "图 X 给出了..." or "如图 X 所示" should be placed as the final sentence of the paragraph immediately before the figure, and the figure should directly follow that paragraph.

## Applicability

- Applies to result-only section drafts.
- Applies to figure placement in rewritten manuscript sections.
- Applies especially to Section 4.3 retargeting drafts.

## Boundary

- This does not prohibit discussion in sections explicitly designated as Discussion.
- If the target journal requires combined Results and Discussion, this preference should be reconciled with the journal format.
