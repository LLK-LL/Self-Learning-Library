# Memory

The user wants equation numbers to align with the right edge of the body text, not with the far page edge.

# Application

For future DOCX equation layout:

- use the manuscript body text width as the equation-table width
- place the equation number in a right-aligned narrow cell at the text right edge
- do not extend formula tables into the page margin
- keep equations visually centred within the body text area
