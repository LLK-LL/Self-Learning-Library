# Same paragraph literature phrasing should avoid repeated templates

In one paragraph, adjacent literature statements should not repeat the same expression pattern when the cited papers use similar methods.

If two studies both use comparable methods, either vary the expression or merge them into one sentence that states the shared method and then distinguishes the study objects, material systems, or conclusions.

This is especially relevant for introduction literature reviews, where too many repeated `某某等采用/通过某方法，研究了...` sentences make the writing read like a mechanical list.
