# DOCX manuscript edits must distinguish partial drafts from full manuscript

When editing DOCX manuscript files in this project, do not assume that the available DOCX is the complete paper.

Files named like `intro_p2_p4` may only contain selected introduction paragraphs and references. If those files are edited and returned as the main result, the user may interpret missing abstract or first introduction paragraph as accidental deletion.

Before editing or finalizing a DOCX, inspect the document structure and confirm whether the abstract, introduction first paragraph, target section, and reference list are present. If any required earlier section is absent, treat the file as a partial draft and either locate the full manuscript or tell the user that the edit applies only to the partial draft.
