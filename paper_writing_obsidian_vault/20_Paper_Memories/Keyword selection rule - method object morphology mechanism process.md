## Memory Content

论文关键词选择应服务于数据库检索、论文定位和摘要主线呈现。对于本项目这类实验-模拟结合的爆炸焊接论文，关键词不应只列具体材料牌号、单一物相或局部结果，而应覆盖论文的五个核心维度：

- 方法维度：说明本文采用的核心技术路线，例如 `SPH-FEM耦合`。
- 研究对象维度：说明材料或结构类别，例如 `复合板`，其检索范围通常比具体牌号更宽。
- 界面形貌维度：说明爆炸焊接过程中的关键几何/形貌特征，例如 `界面波`。
- 组织机制维度：说明影响性能的微观组织或反应产物类别，例如 `金属间化合物`。
- 工艺领域维度：说明论文所属加工方法和学科入口，例如 `爆炸焊接`。

关键词排序可按“方法 -> 研究对象 -> 界面形貌 -> 组织机制 -> 工艺领域”组织，使读者和数据库能够从数值方法、材料结构、界面现象、微观组织和加工工艺五个入口检索到论文。

本项目导师版关键词 `SPH-FEM耦合；复合板；界面波；金属间化合物；爆炸焊接` 的优势在于：它与导师版摘要中的“双主线”一致，即同时覆盖“界面波形成过程”和“金属间化合物尺度预测”；同时避免关键词过度窄化为 `TC4/Al6061` 或 `Al3Ti`，使论文更容易被爆炸焊接、复合板、界面波、金属间化合物和数值模拟相关读者检索到。

## Context

用户询问导师改过的关键词相较于用户本人关键词的优点，并要求总结关键词选择方法写入 Obsidian 论文知识库，且标注为高优先级。

## Potential Generalized Rule

SCI论文关键词应同时承担“检索入口”和“论文主线压缩”的功能。关键词选择时优先覆盖研究方法、研究对象、关键现象、核心机制/组织和工艺/领域，避免全部集中在材料牌号、单一相名称或过窄结果上。若导师给出的关键词能更完整覆盖论文主线，应作为高优先级关键词选择依据。

## Applicability

- 适用于摘要关键词、投稿系统关键词、论文题名和摘要之间的一致性检查。
- 适用于实验-模拟结合、材料加工、界面组织、复合材料和爆炸焊接类论文。
- 适用于判断关键词是否能覆盖论文的主要方法、对象、现象、机制和领域。

## Boundary

- 如果期刊明确要求使用特定主题词、材料牌号或受控词表，应优先满足期刊要求。
- 如果论文的创新点就是某个具体材料牌号或具体物相，则可以保留具体词，但不应牺牲方法、对象、现象和领域入口。
- 关键词数量受限时，优先保留最能代表论文主线和检索价值的词。
