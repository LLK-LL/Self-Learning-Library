# Memory

The user prefers English formula descriptions to use passive voice as much as reasonably possible. Display equations in Word should be visually centred; paragraph text that merely contains inline formula variables should not be centred.

# Application

For future DOCX theory-section edits:

- use passive forms such as "is defined", "is evaluated", "is expressed", "is obtained", and "is introduced" where natural
- keep formula objects intact
- center only standalone display-equation paragraphs
- keep explanatory prose paragraphs justified or aligned with normal body text
