# Memory

The user identified formula-format problems in the manuscript screenshot, including non-centred formulas, incorrect subscripts and superscripts, missing characters, and damaged placeholder boxes. The user also required English descriptions to avoid present perfect tense and use only simple present and simple past.

# Application

For future DOCX formula work:

- inspect the rendered or OMML structure for damaged characters, not only equation numbering
- verify subscripts, superscripts, bars, dots, roots and fractions in each corrected equation
- remove stray inline OMML variables from prose paragraphs when the same variables are already described in text
- check polished English for present-perfect forms such as `has been`, `have been`, `has obtained`, and `have obtained`
