# Memory

The user wants the project working directory to keep only the latest Word manuscript and explicitly retained Word documents. Backup Word files and intermediate Word versions should not accumulate in the working path.

# Application

For future DOCX manuscript work:

- place backup Word files in `word_backups/`
- place intermediate Word versions in `word_backups/` unless the user explicitly asks to keep them in the main working directory
- keep the working directory focused on the latest `manuscript.docx`, the source reference article and user-specified section documents
