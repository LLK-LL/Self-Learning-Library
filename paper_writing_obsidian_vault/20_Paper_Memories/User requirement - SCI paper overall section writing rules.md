# User requirement - SCI paper overall section writing rules

## Memory

The user provided a complete SCI manuscript writing framework and explicitly asked to organize it into an overall paper-writing rule and write it into the project knowledge base writing-rule layer.

The framework requires the manuscript to connect abstract, introduction, theoretical model, numerical simulation, discussion, and conclusion into one coherent research chain. The abstract should state topic, purpose, method, results, and conclusion. The introduction should use recent English SCI literature, usually 10 to 15 papers for a full literature-based introduction, and should proceed from background, to current methods and limitations, to this paper's objectives, content, and conclusions. The theoretical model must explain equations, algorithms, assumptions, and derivations accurately. Numerical simulation should build the model from the theory or algorithm and present clear figures, curves, annotations, and parameters that support the research topic. Discussion should analyze key parameters, mechanisms, comparisons with literature or experiments, engineering application, advantages, limitations, and feasible future solutions. Conclusion should summarize the main work, innovations, advantages over existing methods, limitations, application value, and future extensions.
