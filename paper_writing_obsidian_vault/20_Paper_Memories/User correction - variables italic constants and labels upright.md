# Memory

The user corrected the formula notation rule: variables should be italic, including tensor, vector and scalar variables. Constants and symbols that represent names or labels should be upright.

# Application

For future formula writing:

- italicize variables, including tensor, vector and scalar quantities
- use upright text for constants, function names, abbreviations and label-like identifiers
- do not treat tensors as upright merely because they are tensors
- distinguish variable indices from name/label indices when formatting subscripts and superscripts
- numeric subscripts must be upright
- subscripts that represent abbreviated English words or labels, such as `eff`, `max`, `room`, `melt`, `SF`, `SPH`, `FEM`, or similar name-like identifiers, must be upright
- this typography rule applies both inside displayed/inline formulas and in ordinary prose when symbols with subscripts or superscripts are mentioned
