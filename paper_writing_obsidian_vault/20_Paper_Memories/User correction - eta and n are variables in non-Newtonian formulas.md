# Memory

In the non-Newtonian viscosity formulas, the user corrected that `eta` and `n` are variables in the shown formula and should be italic rather than upright.

# Application

For non-Newtonian fluid formulas, treat the apparent-viscosity symbol and the flow-index symbol as variables unless the user explicitly defines them as fixed constants. Numeric subscripts on these symbols remain upright, but the base variable symbol remains italic.
