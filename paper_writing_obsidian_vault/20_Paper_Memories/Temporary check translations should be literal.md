# Memory

When the user asks for a temporary translation for checking, translate according to the original meaning only. Do not polish, restructure, improve wording, or smooth the academic expression.

# Required Future Behavior

- Treat "checking-use temporary translation" as literal translation.
- Preserve formula tables and equation numbers unless the user asks to modify them.
- Save the translation as a separate Word document.
- Do not turn this type of request into manuscript polishing.
